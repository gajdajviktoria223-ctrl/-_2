# W3C Validator — як швидко зробити протоколи (скріни)

W3C Validator не може напряму перевіряти `file://` URLs, тому найпростіше — **File Upload**.

## Кроки
1) Відкрийте: https://validator.w3.org/
2) Оберіть **Validate by File Upload**
3) Завантажте по черзі:
   - `index.html`
   - `ingredients.html`
   - `recipes.html`
   - `recipe.html`
   - `about.html`
   - `contacts.html`
4) Зробіть скріншот результату
5) Збережіть у `validation/` з назвами:
   - `index-validator.png`
   - `ingredients-validator.png`
   - `recipes-validator.png`
   - `recipe-validator.png`
   - `about-validator.png`
   - `contacts-validator.png`

## Порада
Перед повторною валідацією очистіть кеш браузера або відкрийте у приватному режимі.
