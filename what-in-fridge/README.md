## Сторінки
- `index.html` — головна
- `ingredients.html` — вибір продуктів
- `recipes.html` — список рецептів
- `recipe.html` — сторінка рецепта (параметр `?id=...`)
- `about.html` — про сервіс
- `contacts.html` — контакти (демо‑форма)
