# Що в холодильнику? (What in Fridge)

Навчальний веб‑проєкт: підбір рецептів за наявними інгредієнтами.

## Виконавці
- ПІБ: Гайдай Вікторія Леонідівна
- Група: ЗК-33

## Сторінки
- `index.html` — головна
- `ingredients.html` — вибір продуктів
- `recipes.html` — список рецептів
- `recipe.html` — сторінка рецепта (параметр `?id=...`)
- `about.html` — про сервіс
- `contacts.html` — контакти (демо‑форма)
